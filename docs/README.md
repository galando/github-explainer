# Screenshots

Place `screenshot.png` in this directory.

## Recommended Screenshot

Take a screenshot of the app showing:
1. The hero section with search bar
2. A repository explanation in progress
3. Key features visible (architecture diagram, tech stack, etc.)

### Screenshot Guidelines

- **Size**: 1280x720 or larger
- **Format**: PNG with reasonable compression
- **Content**: Show the app with an interesting repo (e.g., facebook/react)

You can also add:
- `screenshot-dark.png` for dark mode
- `demo.gif` for an animated demo
